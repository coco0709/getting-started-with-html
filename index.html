WWW

World Wide Web (WWW) is a collection of HTML documents and HTML is what glues the web together. The web was invented in 1990 as has over 30 billion pages.

Main componets of the WWW

Computer and browser-Requests are made via the internet to servers using HTTP and the browser displays the files found on the web (EI, Chrome, etc)

The Internet-The worlds largest computer network

HTTP- The main protocol of the web

Servers- Host files that make up the web

HTML

HTML (Hypertext Markup Language) documents form the majority of the content on the web. HTML documents contain text, markup (arrangement/look), references and links. HTML documents are made of HTML elements and the tags tell the browser which element is being used.
